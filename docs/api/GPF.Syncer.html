RequestAuth()

Requests authentication using the current IAuthClient set for this Syncer.

Declaration
public async Task RequestAuth()
Returns
Type Description
System.Threading.Tasks.Task

A Task that completes once the Syncer is authenticated.
Remarks

This method sends an authentication request to the backend, allowing the Syncer to become authenticated before performing further operations that require authentication. The task completes when the authentication process finishes.

Examples
var syncer = Syncer.CreateSyncer("default");
var authClient = new DemoAuthClient(syncer);
syncer.SetAuth(authClient);
await syncer.RequestAuth();
